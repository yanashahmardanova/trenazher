Ниже — 5 ситуационных вопросов для коммерческой службы. В каждом кейсе нужно выбрать наиболее сильный вариант ответа клиенту. Вопросы специально сделаны неочевидными: все формулировки похожи на реальные рабочие письма, поэтому важно учитывать тон, структуру, клиентоориентированность и управляемость следующего шага.

Перед началом заполните данные

Для отправки результатов на почту в версии для GitHub Pages используется EmailJS. Ниже в коде нужно будет подставить свои значения PUBLIC KEY, SERVICE ID и TEMPLATE ID. Без этой настройки тренажёр будет работать, но письма отправляться не будут.

После завершения участник увидит результат на экране, а вы сможете получать его по email.
